# MVP

Minimum viable product

간단한 제품을 만들어서 최소한의 비용으로 제품의 가치를 시장에서 빠르게 확인해 볼 수 있는 방법

```
아이디어 도출 -> 프로토타입 생성 -> 피드백 -> 분석 및 학습
```

일반적으로 이러한 일련의 과정을 반복하면서 가치가 있는 아이디어를 찾거나 아이디어의 가치가 없음을 확인한다. 여기에 소모되는 비용을 최소화 하기 위한 개념이 MVP이다.

* 애자일 방법론과 연관이 있을 것 같다.
