# JSON

JavaScript Object Notation

* ##### 일반적인 JavaScript 객체와의 차이점

  * key가 quotes로 둘러싸여 있다.

  * function을 value로 가질 수 없는 등 value에 제약이 있다.

* ##### JSON methods

  * `JSON.stringify()` : JSON 객체 -> JSON 문자열

  * `JSON.parse()` : JSON 문자열 -> JSON 객체
